## Hướng đẫn cài đặt Demo:
1. Clone Repository
```shell
git clone https://github.com/tinhbui721/GameStore.git
```

2. Thiết lập Startup Project
Mở project -> click chuột phải GameStore.WebUI -> chọn Set as Startup Project

3. Khởi chạy ứng dụng
